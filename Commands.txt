Para guardar las librerias utilizadas en el entorno VENV
pip freeze > requirements.txt

Para cargar las librerias en colaborativo
pip install -r requirements.txt

Crear entorno virtual
python -m venv env

Activar entorno virtual de VENV
.\venv\Scripts\Activate

Actualizar pip
python.exe -m pip install --upgrade pip
